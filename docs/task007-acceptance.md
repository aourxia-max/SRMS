# Task007 收款、分配与预收款验收记录

## 已实现

- `payments`、`payment_allocations`、`prepayment_transactions` 数据模型与 MySQL migration。
- 管理员、超级管理员可登记租金收款；后端使用 JWT 与角色守卫强制校验。
- 未指定账单时按最早到期的未结清账单自动分配；可显式选择未来账单。
- 金额不足时，最后一张账单形成部分付款；金额超出所选账单的部分记入合同预收款流水。
- 收款、分配、账单实收/未收/状态、预收款流水、合同已缴至/下次应缴快照在同一数据库事务内处理。
- 收款页可登记收款、选择账单、查看收款覆盖账期、查看预收款余额和流水。

## 未提前实现

- 收款退款、收款作废及其审批属于后续 Task008。
- 收款时提交的优惠/减免申请及超级管理员确认属于 Task008。
- 押金收取、抵扣和退还属于 Task010；虽保留了冻结枚举，不开放对应业务接口。
