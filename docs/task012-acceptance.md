# Task012 驾驶舱验收记录

## 已实现

- 登录后首页为驾驶舱，采用统一的飞书式浅色、高信息密度界面。
- 支持楼栋与房态多选筛选，显示全部房态数量、房源总数、可经营房源、已出租数和出租率。
- 显示未来 7 天催租、逾期欠租、未来 30 天合同到期和待审批事项。
- 房态图按楼栋、楼层、房号排序，实时展示房间状态。
- 欠租金额汇总仅返回给超级管理员；其他角色仅可见其工作所需的欠租待办数量与明细。
- 首页提供收款登记和合同管理快捷入口。

## UI 基线

- 全局 Element Plus 使用中文 locale，日期/时间选择器显示中文。
- 卡片、表格、表单、按钮和页面内容区已统一为飞书式简洁专业风格。
- 保持高信息密度、少点击、权限清晰和危险操作显式提示的交互原则。

## 验证

- `npm run build`、`npm run lint`、`npm run db:validate` 通过。
- `npm run test` 通过：10 个测试套件、30 个测试。
- `npm --prefix backend run test:e2e -- --runInBand` 通过：1 个测试套件、6 个接口测试。
- 后端健康检查返回 200；未登录访问 `/api/dashboard` 返回 401。
